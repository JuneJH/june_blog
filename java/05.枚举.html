# 枚举

# 1.使用

public enum Hobby {
    football,
    basketball,
    music,
    movie
}
  • 直接利用关键字enum
  • 该枚举自动继承Enum,因此他拥有Enum父类的方法
  • name(): 获取枚举名字
  • ordinal(): 获取枚举的顺序
  • valueof():通过枚举名字获取对应的枚举对象
  • values(): 获取所有枚举对象返回一个数组
  • compareTo(): 比较两个枚举
  • 因此enum不能再继承,可以再实现

# 2. 测试

public static void main(String[] args) {

  System.out.println(Hobby.basketball);//basketball
  Hobby[] hobbies = Hobby.values();
  for(Hobby val : hobbies){
    System.out.println("Hobby"+val.name()+"===>"+val.ordinal());// basketball ---> 0
  }
}

# 3. 自定义属性方法

public enum Hobby {
    football("足球",3),
    basketball("篮球",2),
    music("音乐",1),
    movie("电影",0);

    private String name;
    private int index;
    private Hobby(String name,int index){
        this.name = name;
        this.index = index;
    }
    public String getName(){
        return  name;
    }
    public int getIndex(){
        return  index;
    }
}

# 4. 使用自定义属性

public class Test {
    public static void main(String[] args) {

        System.out.println(Hobby.basketball);
        Hobby[] hobbies = Hobby.values();
        for(Hobby val : hobbies){
            System.out.println("Hobby"+val.name()+"===>"+val.ordinal());
        }
        for(Hobby val : hobbies){
            System.out.println("Hobby:"+val.getName()+"===>"+val.getIndex());//Hobby:足球===>3
        }
    }
}
  1. Enum类中大多数属性都是final修饰不可重写
